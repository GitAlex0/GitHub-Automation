# GitHub-Automation
This is a Python Project, made with Selenium Webdriver!

### How to use?
Clone the Project with
```shell
git clone git@github.com:GitAlex0/GitHub-Automation.git
```
then go to the cloned directory with
```shell
cd Y:\our\path\here
```
and run the main.py with
```shell
py .\main.py
```
